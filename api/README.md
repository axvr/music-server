# Enqueue

_An ethical music streaming service._


## Run server

Start the server:

```
clojure -X:run
```


## Dependencies

- [PostgreSQL](https://www.postgresql.org/)
  - [Install on Fedora](https://fedoraproject.org/wiki/PostgreSQL)
- [libsodium](https://libsodium.gitbook.io/doc/installation)
